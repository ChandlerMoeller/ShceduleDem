Goal: Have an improved app that has the implements of the first UI plus 
modifications based on time matches and event administrator control.

User story 1: As an event scheduler I want to be able to have options to have
options to improve scheduling for volunteer events
	-option to set amount of volunteers needed for a certain time
frame (2 hours)
	-option to set the amount of booths, tables, stations (2 hours)
	-option for how many people needed per booths, tables, 
stations (3 hours)
	Total user story: 7 hours

User story 2: As a volunteer, I want to be able to view the modified schedule
to see when I am needed
	-make XLS Viewer accept modified XLS input (2 hours)
	Total user story: 2 hours

User story 3: As an Event administrator, I want to be able to send the modified
schedule to the volunteers through the app via email
	-UI to input email for each user (1 hour)
	-ability to input email for each user and input into the
modified XLS (4 hours)
	-Send XLS spreadsheets through email popup in app to email
list (3.5 hours)
	Total user story: 8.5 hours
